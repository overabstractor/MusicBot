Política de Privacidad
Última actualización: abril 2025

MusicBot no recopila, almacena ni transmite ningún dato personal a servidores externos. Todo funciona localmente en tu PC.

1. Datos que se almacenan localmente

MusicBot guarda la siguiente información en tu PC, dentro del directorio de la aplicación:

  • Configuración de la aplicación (ajustes, rutas de archivos).
  • Historial de canciones reproducidas y cola persistente.
  • Tokens de autenticación de Spotify, Twitch y Kick almacenados en la base de datos local.
  • Archivos de audio descargados desde YouTube en la carpeta de biblioteca.
  • Registros de actividad (logs) almacenados localmente.

2. Lo que NO hacemos

  • No enviamos ningún dato a servidores propios.
  • No incluimos telemetría, analíticas ni rastreo de uso.
  • No compartimos información con terceros.
  • No requerimos registro de cuenta en nuestros sistemas.

3. Servicios de terceros

Al usar MusicBot, la aplicación se comunica directamente con los servicios que configures:

  • YouTube (via yt-dlp) — para buscar y descargar audio. Las solicitudes se realizan desde tu IP.
  • TikTok, Twitch, Kick — para leer el chat en tiempo real mediante sus APIs oficiales.
  • iTunes Search API — para obtener metadatos y carátulas de canciones.

Cada uno de estos servicios tiene su propia política de privacidad. MusicBot no controla cómo gestionan los datos que reciben.

4. Datos de autenticación

Los tokens OAuth de Twitch y Kick, así como las credenciales de sesión de TikTok, se almacenan únicamente en la base de datos local (musicbot.db) en tu PC. Nunca abandonan tu equipo.

5. Desinstalación

Al desinstalar MusicBot, todos los datos locales (base de datos, archivos de audio, logs) permanecen en tu PC hasta que los elimines manualmente desde el directorio de la aplicación.

6. Contacto

Si tienes preguntas sobre esta política, puedes abrir un issue en el repositorio de GitHub.
