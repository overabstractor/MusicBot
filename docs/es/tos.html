MusicBot es software gratuito y de código abierto. Al usarlo aceptas estos términos.

1. Uso del software

MusicBot es una aplicación de escritorio para Windows que permite gestionar solicitudes de canciones durante transmisiones en vivo. Puedes usarla de forma gratuita, modificarla y distribuirla bajo los términos de la licencia del repositorio.

2. Cumplimiento con servicios de terceros

MusicBot se integra con plataformas externas. Es tu responsabilidad cumplir con los términos de cada una:

  • YouTube — la reproducción de audio se realiza mediante yt-dlp. Debes respetar los Términos de Servicio de YouTube.
  • TikTok — la conexión al chat de TikTok Live está sujeta a los Términos de TikTok.
  • Twitch — el acceso a la API de Twitch está sujeto a los Términos de Twitch.
  • Kick — el uso de la API de Kick está sujeto a sus términos de servicio vigentes.

3. Uso permitido

MusicBot está diseñado para uso personal en transmisiones en vivo. No está permitido usarlo para:
